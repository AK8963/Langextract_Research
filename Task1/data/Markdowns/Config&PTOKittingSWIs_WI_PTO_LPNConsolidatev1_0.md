## **LPN CONSOLIDATE AFTER PTO PICKS**

For the PTO Kitting process (using the Oracle 'Packing Workbench') it is required that all kit components per Kit/Order line are packed into a single LPN. Often however, the actual picks have resulted into multiple LPNs, so these must be consolidated first.

**Step 1:** Sign on to the Oracle Mobile Application Select Responsibility 'ZTC Warehouse System Packer Mobile' Navigate to 'Outbound Packing' -> 'Consolidate'

**Step 2:** Enter the correct Organization and press 'Enter' on the keypad

**Step 3:** In the 'Parent LPN' field either scan a new LPN or one of the existing LPNs you want to consolidate the other LPNs into Press 'Enter' on the keypad

In the 'Next LPN' field scan or enter the first LPN you want to consolidate

![](_page_0_Picture_9.jpeg)

![](_page_0_Picture_10.jpeg)

Select the 'More' button to consolidate more LPNs Select the 'Merge' button if you have scanned all the LPNs that you want to consolidate This will close the window

*Note: At this stage of the process the original LPNs still exist but have the new consolidated LPN as the parent LPN.*

**Step 4:** From the menu select 'LPN Reconfig'

Scan the Parent LPN which you have used to consolidate the LPNs into.

Select the 'Merge-Up' button to complete the consolidation process This will complete the transaction.

*Note: At this stage of the process the original LPNs no longer exist.*

A 'Tx Success' message will be displayed at the bottom of the screen

![](_page_0_Picture_18.jpeg)

The LPNs are now consolidated into a single LPN
