## 1. **OVERVIEW**

This work instruction provides instructions on how to create Serial labels using the Zebra system zDSS.

## 2. **CREATE SERIAL LABEL**

1. Log into Zebra- zDSS system.

![](_page_0_Picture_5.jpeg)

- 2. Scan the password that is displayed on the monitor (Welcomezebra2).
  - ✓ The password changes yearly.
- 3. Click on zDSS shortcut.

![](_page_0_Picture_9.jpeg)

4. In Scan SMART Login, scan the password also display on the monitor (03testmfg).

![](_page_0_Picture_11.jpeg)

![](_page_0_Picture_12.jpeg)

5. Click on 2D Barcode Maker.

![](_page_0_Picture_14.jpeg)

✓ The Printer IP will auto-populate.

✓ If it does not auto-populate enter the IP number on the printer.

![](_page_1_Picture_2.jpeg)

**Ex**:

6. In the Select Label field drop-down menu, select TX5 Carton.

![](_page_1_Picture_5.jpeg)

7. In the Delivery field, enter a generic delivery number.

![](_page_1_Picture_7.jpeg)

- 8. In the Part Number field, enter the SKU number
  - ✓ Remove any extra digits that auto-populate when the Part # is scanned such as (1P).

![](_page_1_Picture_10.jpeg)

![](_page_1_Picture_11.jpeg)

9. Serial Quantity for 2D barcode field, enter the # of serial(s) that will be on the label.

![](_page_1_Picture_13.jpeg)

✓ Max Serial Quantity is 14.

# **CONFIDENTIAL INFORMATION WORK INSTRUCTIONS** Zebra Technologies **CREATE AND PRINT SERIAL LABELS Work Instruction No. Revision No. 01 Site Code WI03 Page 3 OF 4**

10. Enter the COO - Country of Origin.

![](_page_2_Figure_2.jpeg)

![](_page_2_Picture_3.jpeg)

- ✓ COO abbreviations are in Section 3.
- ✓ **Country of Origin cannot be XX.**
- 11. In the Select Format field, from the drop-down menu select the format for the serial # you need.

![](_page_2_Picture_7.jpeg)

![](_page_2_Picture_8.jpeg)

- ✓ A is for the letters on the serial #
- ✓ N is for the numbers on the serial #.
- ✓ The number after the dash is for the # of characters.
- . 12. Click Start

![](_page_2_Picture_13.jpeg)

- ✓ If the part number is scanned instead of the serial # that system will display a Bad Serial # Format error.
- ✓ If the same serial # is scanned the system will not record it and the label will not print.

13. Scan the Serial #(s).

![](_page_2_Picture_18.jpeg)

- 14. The Label Auto prints
- 15. If you have more serial #s of the same format, click Yes at the pop-up.

![](_page_2_Picture_21.jpeg)
