### 1. OVERVIEW

The following procedure assumes that there is no sticker on the analyzer indicating that Zebra Cap Test.cpg has been installed. This procedure takes this opportunity to install this file and ensure that BatteryShop is properly configured.

#### 2. ZEBRACAP TEST (APPENDIX A2)

- **1.** BatteryShop must be installed before ZebraCap Test. If BatteryShop is already installed follow the steps below for ZebraCap Test.
- **2.** Using Notepad or a simple text editor (not Word), create a text file named ZebraCapTest.cpg with the following content and store it on the PC:

- **3.** Open BatteryShop and Click the "Help" menu then "About". Confirm that the BatteryShop version is 7.2 or later.
- **4.** If the BatteryShop is an earlier version proceed to **BatteryShop Installation (APPENDIX A3)**
- **5.** Select "Database" then "Programs", it will display the customs products window.

![](_page_0_Picture_13.jpeg)

**6.** Remove the checkmarks for all the items in the Active window.

![](_page_1_Figure_1.jpeg)

**7.** Click on the Import button and select the **ZebraCapTest.cpg**

![](_page_1_Figure_3.jpeg)

- **8.** Click Save and then the Close icon.
- **9.** Place a sticker on the C7400 that reads, "**ZebraCapTest.cpg** has been installed.
- **10.** You can now proceed with **Cadex Testing** (Appendix A1)
