## **CONSOLIDATE DELIVERIES**

If you have split a PTO Delivery into multiple Deliveries, to allow kitting on multiple PTO Workstations, you will need to consolidate these Deliveries again prior to shipping.

**Step 1:** Using Oracle Forms, navigate to the 'Shipping Transactions Form'

![](_page_0_Figure_6.jpeg)

**Step 4:** First 'Unfirm' the Delivery which allows you to change the contents for the Delivery

Go to 'Actions' and select the option 'Unfirm' and select 'Go' The 'Firm Status' for the Delivery will be updated into 'Not Firm'

![](_page_0_Picture_9.jpeg)

**Step 5:** Go to the 'Contents' tab and select all the LPN lines for the Delivery. Then select the 'Actions' option and 'Unassign from Delivery', followed by 'Go'.

![](_page_0_Figure_11.jpeg)

With the lines still selected, using the 'Actions' option again, select option 'Assign to Delivery'. Enter the original Delivery number into which you want to consolidate the lines and press 'Go'.

If you are unable to find/select the existing (original) Delivery for consolidation, make sure that the status of the lines you have selected is 'Not Firm' and that the Customer and Account Number for both Deliveries match.
