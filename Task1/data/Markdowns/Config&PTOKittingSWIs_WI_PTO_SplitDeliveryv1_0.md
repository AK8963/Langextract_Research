## **SPLITTING A DELIVERY**

If you have a PTO Delivery for which you want to divide the kitting activities over multiple PTO workstations (for example large quantities or multiple, different kits), then you will need to split the Delivery into multiple Deliveries to enable this.

**Step 1:** Using Oracle Forms, navigate to the 'Shipping Transactions Form'

**Step 2:** Select the 'Deliveries' radio button

Enter the Delivery number in the 'From Delivery Name' and 'To Delivery Name' fields

**Step 3:** Make sure you activate the 'Deliveries' Tab

![](_page_0_Picture_9.jpeg)

**Step 4:** First 'Unfirm' the Delivery which allows you to change the contents for the Delivery Go to 'Actions' and select the option 'Unfirm' and select 'Go' The 'Firm Status' for the Delivery will be updated into 'Not Firm'

**Step 5:** Go to the 'Contents' tab and select all lines you want to split off into a new Delivery

Once you have selected all the lines, go to the 'Actions' option again and select 'Unassign from Delivery', followed by 'Go'.

Make sure you select all lines for a Kit, to split these off successfully

![](_page_0_Picture_14.jpeg)

Once the lines are unassigned, you can assign these lines to a new Delivery. With the lines still selected, go to 'Actions' and select option 'Auto-create Deliveries'

The original PTO Delivery is now split into multiple Deliveries to enable kitting on multiple PTO Workstations.
