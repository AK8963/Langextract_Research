# 1. OVERVIEW

The purpose of this instruction is to provide a step-by-step process for using the reel-to-reel device to count labels, thereby speeding up the counting process and making it more accurate and efficient for the picker.

# 2. COUNTER SET UP

- 1. Make sure the lever is set to "STOP".
- 2. Confirm the device screen displays "0/0".
- 3. Preferably, set the speed to the lowest level by adjusting the speed knob.
- 4. Select the desired quantity of labels by pressing the blue buttons from left to right.
- 5. Each button under the screen allows a maximum input of 9, starting from right to left with single units, followed by tens, hundreds, thousands, ten thousand, and hundred thousand.
- 6. To change or decrease the quantity, increase the input value to 9. This will reset the unit to 0, allowing you to set the desired amount.
- 7. To reset the counter, press either the reset button under the display screen or the red counter reset button.
- 8. To lock a specific quantity of labels, select the lock button.

# 3. LABEL ROLL PLACEMENT

- 1. Ensure your roll of labels is placed on the Roll Holder located on the left side of the unit.
  - ✓ The device accommodates rolls with a core inside diameter of 3 inches.
- 2. Use the black knob to adjust tension.
  - ✓ Tightening the knob too much can increase the braking effect (unwind tension).
  - ✓ Loose the knob if the core of the unwind roll is too tight on the hub.
- 3. Feed the label behind the first metal pole and through the sensor.
  - ✓ Labels can be inserted with the print facing either outward or inward.
  - ✓ The sensor will automatically start counting labels as they pass through, counting until they reach the other side of the hub and latch onto the empty core.
  - ✓ The sensor counts each label at its division point.

- 5. Ensure the red lever on the rewinding reel is loose, then insert an empty core label and secure it by tightening the red lever clockwise for stability.
- 6. Attach a label to the empty core on the rewinding reel.
  - ✓ Ensure the tape is applied as close to the edge of the label as possible to prevent damage when removing it.

# 4. COUNTY S EVO COMPONENT COUNTER

- 1. The County S EVO works by counting the holes on the tape. It can operate in two different ways:
  - I. **Totalizer:** components are counted from a zero reference, tape feed is motorized, and the counter automatically stops at the tape end, to prevent loss of the total.
  - II. **Preset mode:** the desired component number is entered on the keyboard, and the counter automatically stops when it reaches the corresponding component. All functions are easy to operate with the help of interactive messages on the display.
- 2. On the screen are always visible:
  - Counting direction
  - Pitch number (holes per component)
  - Working mode (Totalizer, Preset or Rewind)
  - Name of the operator
  - Components counted
  - Date & Time or the last barcode reading
