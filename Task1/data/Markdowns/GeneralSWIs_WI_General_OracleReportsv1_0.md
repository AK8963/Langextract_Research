## WORK INSTRUCTION – GENERAL – ORACLE REPORTS

## **GENERATING ORACLE REPORTS**

There are many different reports that can be generated in Oracle.

This instruction will show how you can generate a report and how to view/print the output.

**Step 1:** From the Toolbar select 'View' -> 'Requests'

The 'Find Requests' screen will be displayed Click on 'Submit a New Request'

**Step 2:** Select the Name of the report you want to generate in the 'Name' field. You can use the standard Oracle search options, for example the List of Values (…), searching using partial names in combination with the wildcard character (%).

![](_page_0_Picture_9.jpeg)

Fill in the required parameters and click on 'OK'

**Step 4:** Click on the button 'Submit'

In the pop-up box select 'Yes' if you want to generate another report, otherwise select 'No'

Once you have selected 'No', click on 'Find' in the pop-up screen 'Find Requests'

**Step 5:** By default all your report requests from the previous 7 days will now be displayed

> For each report you will see both the 'Phase' and 'Status' reported. When these show 'Completed' and 'Normal' the report generation is done successfully and the report is ready for you to view the output

*Note: You can use the 'Refresh' button to refresh the screen.*

To view your report, select it first and click on the 'View Output' button. The report will be displayed, ready for printing
