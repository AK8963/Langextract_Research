## 1. OVERVIEW

The purpose of this work instruction is to show the diagram and required reads when performing daily sensor verification of the CubiScan equipment.

## 2. CALIBRATION PROCESS

- **1.** Ensure CubiScan is plugged in.
- **2.** Test the CubiScan sensor measurements to see if it needs to be calibrated.
  - a) Place the black calibration box with the dimensions *12"x5" x 3.6"* on the CubiScan in the back-left comer as shown correctly below. The output measurements of the CubiScan should read *12" x 5" x 3.6"* with a *1.55lb* weight.
  - b) If test-measured outputs are correct, fill out the *CubiScan Daily Check Form.*
  - c) If the weight is anything other than *1.55 lbs*., move to step 3
  - d) If the dimensions measured are anything other than *12" x 5" x 3.6"*, move to step 4

![](_page_1_Figure_0.jpeg)

- **3.** Calibrate the CubiScan weight scale.
  - a) On the home screen, Select **MENU**
  - b) Select scale option at the bottom and input the weight of the calibration cube, which is 1.55 llbs
  - c) Select **Next**
  - d) Clear the glass scale platform of all objects.
  - e) Select **Next**
  - f) Place the calibration cube on the scale.
  - g) The CubiScan scale is now calibrated
- **4.** Calibrate the CubiScan sensor measurement reading.
  - a) On the home screen, Select **MENU**
  - b) Select the sensors option at the bottom and make sure the swing gate is in the home position on the right.
  - c) Select **Next**
  - d) Clear the glass platform of all objects

- e) Select **Next**
- f) Place the calibration cube on the platform and slide it against the back corner until it is in contact with both side panels with the 12" side against the left panel with the 5" end side touching the right panel.
- g) Select **Next**
- h) Turn the calibration cube so it is standing 12" tall with both end sides touching back panels.
- i) Select **Next**
- j) The CubiScan sensor measurements have now been properly calibrated.
- k) Select the Home button.
- **5.** Once calibrated, fill out the *CubiScan Daily Check Form*

# 3. VERIFICATION READINGS FOR L, W,& H

- **1.** Place the case in the back corner of the CubiScan
  - The long side of the case should be aligned against the back-left wall.
- **2.** The weight is automatically measured and displayed in the control panel.
  - ✓ Do not lean on the glass or have anything but the item on the CubiScan to receive an accurate weight.
- **3.** Select the *Measure* button from the control panel.
- **4.** Irregular and non-cube-shaped items must be measured by rotating the arm over the
