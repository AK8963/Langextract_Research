## **LABEL PRINTING (PTO)**

The Labels used in the PTO Kitting process are all automatically triggered as part of the process.

If a PTO Kit or Overpack Label is damaged (or missing), the label can be reprinted from the Label Printing option on the scangun (MWA).

The following PTO labels can be generated:

| Label<br>Type | Format            | Label Description                                             | Preview |
|---------------|-------------------|---------------------------------------------------------------|---------|
| LPN           | EU_KIT_LABEL      | Kit label for the individual kit                              |         |
| LPN           | EU_KIT_BABY_LABEL | Small kit label for the individual kit – with limited<br>data |         |
| LPN           | EU_OVERPACK_LABEL | Overpack label for multiple kits packed into a<br>single LPN  |         |

**Step 1:** Using the Mobile Application, select the Responsibility 'ZTC Warehouse System Packer Mobile' and navigate to 'Labels'

**Step 2:** Select option 'Printer Sign-on' to log into the printer

- Scan or enter the printer's name
- Select 'All Labels' as the Label Type

Click on the 'Sign On' button

The message 'Sign on successful' will be displayed at the bottom

*Please note: If you are already signed on to another printer, then use* 

*the 'Sign-Off' option first*

![](_page_0_Picture_16.jpeg)

*Note: To login to the printer for the PTO labels, you only have to complete steps 1 and 2*

**Step 3:** Select option 'Label Request' to print a label

- Select the correct Label Type (using the search icon)
- Select the correct Format (using the search icon)
- LPN: Enter the Kit LPN

*Note: You do not have to scan the printer's name again*

- Enter the number of labels ('Copies') you want to print (default = 1)

Click on 'More' to print more labels or 'Done' if you are finished printing

The labels can also be reprinted from the 'Label History' program in Oracle Forms.

**Step 1:** Using responsibility 'ZTC Warehouse System Packer' navigate to 'WMS Inquiries' -> 'View Label Requests'

![](_page_1_Picture_4.jpeg)

## **Step 2:** Enter the selection criteria:

- User: Enter the user ID of the person that originally printed the label (optional)
- Printer
- Label Type: 'LPN'
- Label Format: Enter the correct Label format:
  - o Product Label: 'EU\_KIT\_LABEL' o Pack Label: 'EU\_KIT\_BABY\_LABEL' o Overpack Label: 'EU\_OVERPACK\_LABEL'
- Request Date: Select the date the labels were originally Printed

*Please note: Under the 'Label Info' tab there is also an option to specify the item number or an LPN number* 

Click on the 'Find' button

![](_page_1_Picture_14.jpeg)

![](_page_1_Picture_15.jpeg)

**Step 3:** The system will now show the list of labels that can be reprinted

## *Please note:*

*In the details under 'Label Content' you can find Kit number or Overpack number*

Select the line with the label that you want to reprint There is no need to enter the printer or quantity, the reprint will always be done to the original printer
