## **PTO – KIT CREATION**

The PTO Kitting process – using the Oracle Packing Workbench – covers the process of creating the customer-specific kits. This kitting process can commence once the materials have been picked and the PTO Stations have been set up for kitting.

The PTO Kit Sheet (part of the Pack Consolidation Report) shows all the required information for the kits:

![](_page_0_Figure_6.jpeg)

![](_page_0_Picture_7.jpeg)

Please double-check that you have all materials required (packed into a single LPN) and that the Country of Origin is updated against the Delivery in case of a multis-sourced item (this is highlighted on the Pack Consolidation Report) Also ensure that you are logged into the Kit (or Baby) Label printer located on the PTO Station you will be using. Before you start kitting, please ensure that the file is displayed in your task bar.

**Step 1:** Use responsibility 'ZTC Warehouse System Packer' And navigate to 'Packing Workbench'

Enter the correct Organization and press 'OK'

**Step 2:** Scan the barcode for the Packing Station you are working on

![](_page_0_Picture_12.jpeg)

Then scan the Delivery number from the PTO Kit Sheet

*Note: By default, the Packing Process option is set to 'LPN Split' which is the correct option for creating the kits*

Click on 'Find'

![](_page_0_Figure_16.jpeg)

**Step 3:** The Packing Workbench will be displayed Some important features:

![](_page_0_Picture_18.jpeg)

- Packing Message: this message indicates the next step in the kitting process
- Content Information: details of current scan
- LPN Content: This section shows the kit components already scanned into the kit
- Eligible Material: This section shows the items, LPNs and quantities of the materials related to the kitting process

**Step 4:** To complete a kit, perform the following scans:

- 1. First scan the LPN that contains all the kit components
- 2. Then scan each kit component. It is best to start with the non-serialized items.
  - o Scan the item barcodes for the non-serialized components from the PTO Kit Sheet and place the components inside the box (kit)
  - o Scan the Serial Number for any serialized component from the label on the item and place the component inside the box (kit)
    - *Note: if there are no serialized items you can skip this step*

![](_page_1_Picture_8.jpeg)

The quantity of each component in a kit is not always '1', so please check the PTO Kit Sheet for the quantity per Kit

*Note: The Packing Message, Content Information, LPN Contents and Eligible Material details will be updated after each scan.*

**Step 5:** Once all the components for a single kit have been scanned, the kit must be closed, and the label must be generated and applied.
