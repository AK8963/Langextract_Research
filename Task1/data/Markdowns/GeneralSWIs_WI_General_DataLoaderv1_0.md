WORK INSTRUCTION – GENERAL – DATA LOADER

## **ENTERING DATA USING THE DATA LOADER TOOL**

Whenever there is a need to enter large amounts of data into an Oracle Desktop Form, the Data Loader Tool can be used instead of manually keying in the data.

The 'Data Loader ' Tool can load the data from an Excel or CSV files into Oracle.

The Data Loader is available in 2 versions:

- Classic (Free version) This can be used from any computer to load the data. Whilst loading, it is not possible to use your computer for other tasks. Even pop-ups from e-mail or Teams may affect the data loading. This version can be downloaded fro[m https://www.dataload.com/](https://www.dataload.com/)
- Professional version This version has more features, in particular improved error handling and the ability to run the upload in the background. This version can be ordered via<https://www.dataload.com/>
