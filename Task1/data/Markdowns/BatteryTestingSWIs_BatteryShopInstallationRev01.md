## 1. OVERVIEW

The following procedure assumes that there is no sticker on the analyzer indicating that Zebra Cap Test.cpg has been installed. This procedure takes this opportunity to install this file and ensure that BatteryShop is properly configured.

# 2. BATTERYSHOP INSTALL (APPENDIX A3)

- **1.** Locate and double-click the BatteryShop.
- **2.** Install file Batshop\_7.1.1.0.exe.
  - ✓ This will unzip the needed files to a temporary folder and get ready for installation onto the computer.

![](_page_0_Picture_8.jpeg)

- **3.** Once the files are unzipped, click Next to run the Setup option for the installation.
  - ✓ The Wizard will walk you through the setup of BatteryShop.

![](_page_0_Figure_11.jpeg)

**4.** Enter the information below and click Next.

![](_page_0_Picture_13.jpeg)

*This document contains proprietary information which is the exclusive property of Zebra Technologies Corp.*

- **5.** You should see an installation complete message. Click Finish
  - ✓ You might see version 7.2 rather than 7.1, and that's ok.

![](_page_1_Figure_2.jpeg)

**6.** Click Next on the Battery Setup Wizard message.

![](_page_1_Figure_4.jpeg)

- **7.** With the C7 connected, identify the Comm Port that it's installed on by using Windows Device Manager and opening Ports (Com & LPT).
- **8.** Click on Ports Map to select (COM6) and click Next.

![](_page_1_Figure_8.jpeg)

- **9.** The next message will now ask you to set up the default Battery ID printer.
  - ✓ Since we are not printing labels or Service Reports, click Next.

![](_page_2_Figure_2.jpeg)

**10.** Click Next on the Dymo label printer and on the Service Report printer messages.

![](_page_2_Figure_4.jpeg)

![](_page_2_Figure_5.jpeg)

**11.** Enter the Service period of 6 months and click Next.

![](_page_2_Figure_7.jpeg)

**12.** Set the options below and click Next.

![](_page_2_Picture_9.jpeg)

*This document contains proprietary information which is the exclusive property of Zebra Technologies Corp.*

**13.** Click Finish on the Complete Setup message.

![](_page_3_Figure_1.jpeg)

**14.** Execute **ZebraCapTest.cpg (Appendix A2)** procedure.
