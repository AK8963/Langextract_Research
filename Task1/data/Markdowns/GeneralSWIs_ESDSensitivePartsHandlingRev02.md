## 1. **OVERVIEW**

The following work instruction describes the process of handling ESD-sensitive parts. The workflow is to be followed by the TX5 material pickers.

# 2. **GENERAL ESD HANDLING GUIDELINES**

1. The ESD components can be identified by the following labels

![](_page_0_Picture_5.jpeg)

![](_page_0_Picture_6.jpeg)

![](_page_0_Picture_7.jpeg)

2. Additionally, the ESD-sensitive components can be identified by the following packaging

![](_page_0_Picture_9.jpeg)

- ✓ When the part looks like an ESD-sensitive part, it should be handled like one.
- ✓ Majority of ESD-sensitive components can be distinguished by an exposed PCB board.

![](_page_0_Picture_12.jpeg)

3. When picking the ESD components from the OEM packaging, the associate needs to bring the whole case to the ESD station and put the grounding wristband on prior to handling the units.

![](_page_1_Picture_2.jpeg)

- ✓ The green light on the ESD monitoring device will indicate if the associate and workstation are grounded.
- 4. The unbagged components should never touch each other during handling.
- 5. The ESD components need to be packaged in static dissipative/protective packaging prior to leaving the ESD station.
- 6. Every station should be cleaned using a non-silicone cleaner.

**1.** The bulk pack ESD items can be identified as parts packaged in the OEM box/bag without individual trays/dividers.

# 4. **PARTS PACKAGED IN TRAYS/LAYERS**

**1.** The ESD items packaged in trays/layers can be identified as the items packaged in the OEM box, separated in trays or carton layers (multiple items or a single item per a tray/layer slot)

![](_page_3_Picture_6.jpeg)

![](_page_3_Picture_7.jpeg)

![](_page_3_Picture_8.jpeg)

**2.** When picking a partial quantity (less than a full case), the associate can open the box to peek inside.

**3.** Whenever possible, the parts should be picked and secured in a tray/layer.

![](_page_4_Picture_2.jpeg)

**4.** If picking whole trays/layers is not possible (the pick quantity is less than the tray/layer quantity, or parts can't be secured in a tray/layer) and the parts don't have individual packaging from OEM, the associate needs to place the contents of each tray/layer slot in individual poly bags. **5.** If a part is serialized, print, and attach the 2x2. Refer to Create and Print TX5 Parts Labels for each individual small poly bag.
