## **PTO WORKSTATION SET UP**

The PTO Kitting process, including the creation of the Overpacks, is done using the Oracle 'Packing Workbench' functionality.

The process has been designed to allow the users to scan barcodes for all the actions they need to do to create a Kit or an Overpack.

Seeing that the entire process is done via scanning, it is required to have the computers enabled for this.

Before you start kitting, please ensure that the 🔢 file is displayed in your task bar. 🔞
