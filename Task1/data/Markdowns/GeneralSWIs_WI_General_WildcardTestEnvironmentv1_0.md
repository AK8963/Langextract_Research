WORK INSTRUCTION – GENERAL – WILDCARD ENVIRONMENT

## **ORACLE WILDCARD ENVIRONMENT**

The Wildcard environment is a weekly copy from the Production environment. It is refreshed every weekend, so it contains all the Production data from the week before.

It is a great environment for testing and trying things

## *Please note:*

*It is not linked to other applications, such as Siebel, Infoship etc.* 

## **1.1 ORACLE FORMS**

You can access the Wildcard environment from the following link:

[https://wildcard-erp.zebra.com:4427/OA\\_HTML/AppsLocalLogin.jsp](https://wildcard-erp.zebra.com:4427/OA_HTML/AppsLocalLogin.jsp)

This is a non-SSO link, so you have to enter a password.

Your password is either 'welcome1' or the same as your password for the Mobile Application (MWA) in the Production environment.

## **1.2 MWA**

Please use the following settings to access the Wildcard Mobile Application

- Site = Wildcard
- Hostname = z77st-erpapp01b
- Port = 11016
